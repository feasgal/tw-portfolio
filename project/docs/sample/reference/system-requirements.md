# System requirements

Use supported OS versions and hardware that meets your protected apps' needs.

## Operating systems

-   Windows: all releases currently supported by Microsoft.
-   macOS: all releases currently supported by Apple.
-   Linux: not supported unless a product vendor states otherwise.

## Network

-   Reliable LAN for server and clients.
-   Bonjour/mDNS for discovery on public servers.
-   Open TCP port used by the server.
