# Server Admin settings

All fields available in the server UI. Names may vary slightly by version.

## Identity and access

-   **Server name**: label shown to clients.
-   **Require password**: force authentication for every connection.
-   **Require client identity**: collect user/machine info.

## Discovery

-   **Advertise on local network**: enable discovery.
-   **Public / Private**: toggle broadcast vs manual connect.
-   **Port**: TCP port used for client connections.

## Licensing

-   **Active licenses**: seats allocated to the server.
-   **Seat timeout / lease period**: how long idle clients keep a seat.
-   **Max leases per client**: cap concurrent usage.

## Monitoring

-   **Status**: online/offline, uptime.
-   **Statistics**: active clients, leases over time, denials.
-   **Logs**: connection and error logs with timestamps.

## Files

-   **Export client configuration**: generate a file for private servers.
-   **Backup / restore settings**: save and load server config.
