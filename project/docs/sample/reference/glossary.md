# Glossary

**Account**

:    Your identity for Acme License services.

**Acme dongle**

:    Hardware key that holds licenses.

**Acme License Engine**

:    System components required by the Hub and protected apps.

**Acme License Hub**

:    Desktop app for managing your license activations.

**Activation**

:    Moving a license to a location.

**Cloud session**

:    A signed-in state that makes Cloud-enabled licenses available.

**Client**

:    A workstation that connects to a server for seats.

**Client configuration file**

:    A file that defines how to reach a private server.

**Deactivation**

:    Returning a license to your account.

**Discovery**

:    How clients find public servers on the LAN.

**Lease**

:    A time-bound grant for a client to use a seat.

**License**

:    A product entitlement in your account.

**Private server**

:    A server that does not advertise on the LAN.

**Public server**

:    A server that advertises on the LAN for auto-discovery.

**Seat**

:    One concurrent use of a license.

**Server**

:    The service that grants seats to clients.

**Server Admin**

:    The admin UI for the server.

**Synchronization**

:    Refresh action for an Acme dongle to fix state issues.
