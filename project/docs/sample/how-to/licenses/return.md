# File a Return Authorization

Replace a broken, lost, or stolen Acme dongle through Acme’s service process.

!!! goal

    Submit a Return Authorization request for hardware replacement.

## Before you start

Gather the dongle’s serial number and any proof-of-purchase your support team might request. If the dongle still appears in the Hub, try a quick **Synchronize** first.

## Steps

1. Sign in to your account on the Acme website.
2. Open **Support** → **Returns/RA**.
3. Choose **Broken**, **Lost**, or **Stolen**.
4. Fill in the requested details and submit the request.
5. If instructed, package and ship the device to the provided address.

!!! outcome

    You receive confirmation and tracking for the Return Authorization. Replacement processing begins per the selected case type.
