# Start or end a Cloud session

Make Cloud-enabled licenses available on this machine while you’re signed in. Use this when you don’t have a dongle attached or prefer cloud-based access.

!!! goal

    Open or close an Acme License Cloud session in the Hub.

## Before you start

Sign in to **Acme License Hub** and confirm your account includes Cloud-enabled licenses. Ensure the machine has a stable internet connection.

## Steps

1. To **start**: in the Hub menu, choose **Open Cloud Session** and wait for the status to show **Open**.
1. If the license for your protected app doesn't automatically move to the cloud session, [activate it there](/sample/how-to/licenses/activate.md).
1. Run your protected app while the session remains open.
1. To **end**: return to the Hub and choose **Close Cloud Session**; confirm the status shows **Closed**.

!!! outcome

    While the session is **Open**, Cloud-enabled licenses are available to your apps. Closing the session releases them.
