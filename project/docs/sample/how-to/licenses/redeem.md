# Redeem an activation code

Claim a new license by adding it to your account with the code provided by your vendor.

!!! goal

    Add a new license to your account using an activation code.

## Before you start

Sign in to **Acme License Hub** and have the activation code ready.

## Steps

1. In the Hub, choose **Redeem**.
1. Enter the activation code exactly as provided.
1. Submit and confirm the license appears under **Available**.

!!! outcome

    The license is added to your account and ready to activate to a location.
