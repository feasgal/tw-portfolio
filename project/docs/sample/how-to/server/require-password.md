# Require a server password

Protect your license server by requiring a password for every client connection. Use this when your server is publicly discoverable on the LAN or whenever you want to prevent unauthorized access, even on private servers.

!!! goal

    Enforce an access password so only authorized clients can connect.

## Before you start

Sign in to **Server Admin** with admin rights.

## Steps

1. Open **Server Admin**.
2. Go to **Access control**.
3. Enable **Require password**.
4. Set and confirm a strong password.
5. Apply changes.

!!! outcome

    New and existing clients must authenticate with the access password before connecting.
