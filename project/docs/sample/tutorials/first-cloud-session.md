# Start your first Acme License Cloud session

**Time:** ~5 minutes  

!!! example "What you'll accomplish"

    Open an **Acme License Cloud** session in the Hub and run a protected app while the session is active.

!!! checklist "You need"

    - Acme License Hub installed
    - Internet access
    - Cloud-enabled licenses in your account

## Step 1: Sign in

1. Open **Acme License Hub**.
2. Sign in with your Acme account.

## Step 2: Open a Cloud session

1. In the Hub menu, choose **Open Cloud Session**.
2. Wait for the status to update.

**The session status now shows Open and Cloud-enabled licenses are in it.**

## Step 3: Run a protected app

1. Launch your protected app while the session is open and confirm that it runs without licensing errors.

## Step 4: Close the Cloud session

1. Return to the Hub.
2. Choose **Close Cloud Session**.

**The session status now shows Closed.**

!!! success

    The app runs while the Cloud session is open. Closing the session releases all licenses.

## Troubleshooting

- Sign-in or connectivity errors → see **[Acme License Hub errors](/sample/troubleshooting/hub-errors.md)**.
- App still shows a license error → in the Hub, confirm your license is Cloud-enabled and that the Cloud session status is Open.
