# Install Acme License Engine and Hub

**Time:** ~15 minutes  

!!! example "What you'll accomplish"

    Install the Acme License Engine components and the Acme License Hub, then sign in.

!!! checklist "You need"
    - Administrator rights on the machine
    - Internet access
    - An Acme account

## Step 1: Get the installer

1. Download the latest **Acme License Engine** installer for your OS from the official site.
2. Verify the file finished downloading before you proceed.

## Step 2: Prepare the system

1. Close all licensed apps.
2. Save work and quit other installers or updaters.

## Step 3: Install Engine and Hub

!!! tip inline end

    If your OS blocks components the first time, open Security & Privacy and click **Allow**, then rerun the installer.

1. Run the installer with admin privileges.
2. Select **Acme License Engine** and **Acme License Hub**.
3. Accept any security prompts.
4. Finish the install and restart the machine if prompted.

## Step 4: Sign in to verify

1. Open **Acme License Hub**.
2. Sign in with your Acme account.
3. Confirm you can reach your account data.

!!! success

    The Hub shows your account name and no connectivity errors.

## Troubleshooting

- Installer completed but the Hub won’t open or errors appear → see [Engine and Hub errors](/sample/troubleshooting/hub-errors.md).
