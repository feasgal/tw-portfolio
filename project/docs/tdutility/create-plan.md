# How-to: Import or create a TDU document plan

Use this procedure to set up or update the document plan for a project. You can either import an existing plan from a spreadsheet or build it directly in TDU.

!!! goal

    Populate the **Document Plan** tab with the documents, types, and assignments for the project.

## Before you start

Confirm that you have:

-   An initial document plan in Excel or another planning tool, or a clear list of documents you need to create.
-   The agreed column mapping if you plan to import from `.xlsx`, `.xls`, or `.csv`.
-   Author identifiers in `firstname.lastname` format for any documents you plan to assign.

## Steps

### Option A. Import a plan for a larger project

1.  Open the plan in Excel or your planning tool.
1.  Export the plan to `.xlsx`, `.xls`, or `.csv` using the column mapping agreed with your tools team.
1.  In TDU, open the project and go to the **Document Plan** tab.
1.  Import the exported file.
1.  Review the imported entries:

    -   Confirm titles, types, and statuses look correct.
    -   Confirm author assignments use the expected identifiers.
1.  Fix any mapping or formatting issues you see, then save the project.

### Option B. Create or extend a plan for a smaller project

1.  In TDU, open the project and go to the **Document Plan** tab.
1.  Right-click in the document list.
1.  Choose **Add Chapter**, **Add Subject**, or **Add Task** as needed.
1.  For each new entry, enter the document details, including title, type, any author assignments, and status.
1.  Repeat for each document you need to add.
1.  Save the project.

!!! outcome

    The **Document Plan** tab lists the planned documents for the project with correct types, statuses, and any initial assignments.
